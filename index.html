Welcome to Safe Social

What is Safe Social?

Safe Social is a web app that allows users to
create events at municipality locations and
send invitations to guests to attend in a COVID-safe manner.

Features

Guest Coordination
COVID Tracking
Guest Communication
Virtual Ticket
